# Internet Radio Player
---
I have a very mild fascination with internet radio so I wrote this player to learn a bit of React. The UI is somewhat inspired by software like Winamp and [modplug player](https://openmpt.org/legacy_software).   

## Installing Dependencies
Run ```npm install``` to install packages. 
## Running Dev Server
Run ```npm run dev``` to start the development server. 
## Future Goals:
- Volume Slider
- Grabbing stream information from the URL and displaying it in the player.
- Scrolling text (like in Winamp).
